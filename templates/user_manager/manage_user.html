Viewer (Visor)

Este nivel tiene acceso de solo lectura. Los usuarios pueden ver información de colas y la lista de respaldos, pero no pueden acceder a los datos de respaldo ni a información sensible.

Backup Operator (Operador de Respaldos)

Además de los permisos de nivel Visor, los usuarios en este nivel pueden ver los respaldos de los equipos e iniciar respaldos instantáneos.

Host Manager (Gestor de Equipos)

Los usuarios de este nivel pueden configurar, añadir y eliminar equipos, y borrar respaldos. También cuentan con todos los permisos del nivel Operador de Respaldos.

Configuration Manager (Gestor de Configuración)

Además de los permisos de nivel Host Manager, los usuarios de este nivel pueden configurar perfiles de respaldo, llaves SSH y grupos de equipos.

Administrator (Administrador)

Los usuarios de este nivel tienen derechos administrativos totales, incluyendo todos los permisos de los niveles anteriores.
